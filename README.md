## What is this?

Mesh VPN System is a peer-to-peer virtual private network that enables secure, direct communication between nodes without requiring manual port forwarding. It uses modern NAT traversal techniques (such as ICE/STUN) to establish connections across different networks, forming a decentralized mesh where each node can communicate with others securely. The system includes a signaling server for coordination and leverages WireGuard-like principles for encrypted tunneling between peers.
